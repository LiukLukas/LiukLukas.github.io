VS CODE

Visual Studio Code - toksai teksto editorius, skirtas įvairiems programuotojams. Atstoja kompaktišką programavimo aplinką, yra geras, greitas, palaikantis įvairias kalbas ir su viskuo suderinamas, puikus nemokamas editorius. Nepatikėsite, bet tai tikrai Microsoft produktas ir tikrai geras.

Kodėl Visual Studio Code toksai geras - tai gal todėl, kad tai yra tiesiog forkas nuo pagarsėjusio Atom editoriaus, kurį sukūrė GitHub kompanija, kurią, savo ruožtu, nusipirko Microsoft. Ir kažkodėl netgi padarė taip, kad Atom neuždarė, o paliko toliau egzistuoti ir vystytis, o jų kloną tiesiog pasiėmė pas save ir irgi ėmė vystyti nepriklausomai. Tai dabar ir yra du baisiai panašūs, vos ne vienodi, bet visgi savybėmis besiskiriantys editoriai - šitas Visual Studio Code ir Atom. Ir sunku net pasakyti, kuris iš jų yra geresnis.

Tai rekomenduojame ar vieną, ar kitą, nes abu visai geri.
